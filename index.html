How to Use This Tool

Recommended setup order:
  1. Go to Settings and fill in project information.
  2. Go to Import Data → Registers and import your SOV, Work Packages, Cost Codes, and Phase Codes.
  3. Go to Import Data → Estimate Inputs and import your Quote, Material List, Labor Hours, etc.
  4. Go to Import Data → Current Spend and import PO logs, invoice logs, labor reports, etc.
  5. Go to Review & Classify to review and approve spend classifications.
  6. View Comparisons and Dashboard for cost analysis.

Important: Only approved spend items affect the official EAC and variance calculations. Low-confidence or unreviewed items are flagged on the Dashboard but do not move the numbers until approved.
