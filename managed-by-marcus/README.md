# Managed by Marcus

A browser game built in React + TypeScript + Tailwind CSS.

## How to run

1. Install Node.js and npm.
2. In the `managed-by-marcus` folder, run:
   ```bash
   npm install
   npm run dev
   ```
3. Open the local Vite URL shown in the terminal.

## Run tests

From the project folder:
```bash
npm test
```

For interactive watch mode:
```bash
npm run test:watch
```

## Game concept

- Control Marcus with `WASD` or arrow keys.
- Choose the meeting-based answer to win.
- `25` meetings = victory.
- `3` real solutions or low credibility = loss.
